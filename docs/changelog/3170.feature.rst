Drop support for Python 3.8; virtualenv now requires Python 3.9 or later to run and to create environments. Remove the
embedded ``wheel`` seed package, which virtualenv bundled only for Python 3.8. The ``--wheel`` and ``--no-wheel`` options
stay as no-ops, but now warn that virtualenv will remove them in a release after 2026-12 - by :user:`gaborbernat`.
